# BALLOONICE Design System

## Colors
- Gold: #D4AF6A (primary accent, CTAs)
- Cream: #F5ECD7 (backgrounds, cards)  
- Rose: #E8B4A0 (secondary accent)
- Dark: #2C1810 (text, headers)
- BG Light: #FFFDF9 (main background)
- BG Cream: #FAF6F0 (section alternates)

## Typography
- Display: Heebo Black (Hebrew headlines)
- Body: Heebo Regular (Hebrew text)
- Accent: Playfair Display (English/luxury feel)

## Spacing Scale
4, 8, 12, 16, 24, 32, 48, 64, 96, 128

## Animation Principles
- Spring easing: [0.23, 1, 0.32, 1]
- Reveal duration: 600-800ms
- Stagger: 80-120ms between elements
- Parallax speeds: 0.3x, 0.5x, 0.7x

## Component Library

### Shared Components
- FloatingBalloons - Decorative animated balloons
- ShimmerText - Gold shimmer text effect
- Navbar - Fixed navigation with logo

### Section Components
- Hero - Full viewport intro with CTA
- Services - Service offerings grid
- About - Company story section
- Testimonials - Customer reviews carousel
- Contact - Contact form + info
- Footer - Links + social

## RTL Considerations
- All text aligned right by default
- Flex direction reversed where needed
- Icons positioned on right side
- Numbers remain LTR
